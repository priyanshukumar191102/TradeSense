# TradeSense

AI-powered investor decision support system using machine learning, technical indicators, and backtesting for Indian stock markets.

## Project Overview

TradeSense is a data science and business intelligence project designed to generate Buy, Hold, and Sell recommendations for Indian stocks. The project combines technical analysis, machine learning, backtesting, and interactive dashboards to support investment decision-making.

## Objectives

* Analyze historical Indian stock market data
* Generate technical indicators
* Train machine learning models
* Produce Buy/Hold/Sell recommendations
* Backtest investment strategies
* Visualize insights using Power BI

## Research Questions

1. Can technical indicators predict short-term investor actions?
2. Which indicators contribute most to Buy/Hold/Sell decisions?
3. Can machine learning outperform simple trading strategies?
4. Would following TradeSense recommendations improve investment outcomes?

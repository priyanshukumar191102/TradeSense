## Entry 1 – Project Initialization

Date: June 2026

### What is TradeSense?

TradeSense is an AI-powered investor decision support system that generates Buy, Hold, and Sell recommendations for Indian stocks using technical indicators and machine learning.

### Initial Project Decisions

* Market Focus: Indian Stocks
* Prediction Horizon: 5 Trading Days
* Labels:

  * BUY: Future Return > 3%
  * SELL: Future Return < -3%
  * HOLD: Everything Else

### Planned Technical Indicators

* SMA
* EMA
* RSI
* MACD
* Bollinger Bands
* Volatility
* Returns
* Volume Indicators

### Machine Learning Models

* Logistic Regression
* Random Forest
* XGBoost

## Entry 2 – Historical Stock Data

What I learned:

- Historical stock data contains Open, High, Low, Close and Volume values.
- Each row represents one trading day.
- Reliance data from 2015–2025 contains 2716 observations.
- Data quality is important before creating indicators or training models.
- Adjusted prices help account for stock splits and corporate actions.
